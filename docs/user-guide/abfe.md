# Absolute Binding Free Energy

Absolute binding free energy (ABFE) workflows estimate the binding free energy of one ligand or molecular partner against a target system.

Use this page for ABFE-specific setup notes, recommended input organization, production commands, and analysis guidance.

## Starter Checklist

- Prepare validated receptor and ligand structures.
- Define the binding pose and simulation box.
- Review force field choices and ligand parameters.
- Run the ABFE preparation workflow.
- Inspect logs and analysis outputs before using results downstream.
