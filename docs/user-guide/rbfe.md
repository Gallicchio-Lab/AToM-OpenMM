# Relative Binding Free Energy

Relative binding free energy (RBFE) workflows estimate free energy differences between related ligands, poses, or molecular variants.

Use this page for RBFE-specific setup notes, perturbation planning, production commands, and analysis guidance.

## Starter Checklist

- Choose a clear reference and target state.
- Validate atom mappings or coordinate swapping inputs.
- Prepare both bound and unbound legs as required.
- Run production with appropriate replica exchange settings.
- Compare analysis outputs across the perturbation network.
